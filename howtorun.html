C++ Compilation and Execution

Step 1: Save Your Code

Save your C++ code with a .cpp extension (e.g., program.cpp)

Step 2: Open Terminal/Command Prompt

Windows: Open Command Prompt or PowerShell

Linux/Mac: Open Terminal

Navigate to the folder containing your .cpp file using cd command

Step 3: Compile the Program

g++ -o program program.cpp

Explanation:

  • g++ - The C++ compiler
  • -o program - Creates executable named "program"
  • program.cpp - Your source code file

Step 4: Run the Program

./program

On Windows:

program.exe

Alternative: One-Step Compilation and Execution

g++ program.cpp && ./a.out

This compiles and runs in one command (creates default executable a.out)

Common Compiler Flags

g++ -Wall -std=c++17 -o program program.cpp

Useful flags:

  • -Wall - Enable all warnings
  • -std=c++17 - Use C++17 standard
  • -g - Include debugging information

Java Compilation and Execution

Step 1: Save Your Code

Save your Java code with a .java extension

Important: File name must match the public class name

Example: If your class is HelloWorld, save as HelloWorld.java

Step 2: Open Terminal/Command Prompt

Windows: Open Command Prompt or PowerShell

Linux/Mac: Open Terminal

Navigate to the folder containing your .java file using cd command

Step 3: Compile the Program

javac HelloWorld.java

Explanation:

  • javac - The Java compiler
  • HelloWorld.java - Your source code file
  • This creates HelloWorld.class (bytecode file)

Step 4: Run the Program

java HelloWorld

Important Notes:

  • Use java (not javac) to run
  • Use class name (not file name with .class extension)
  • Case-sensitive: HelloWorld not helloworld

Complete Example

1. Create file: Student.java

public class Student {
    public static void main(String[] args) {
        System.out.println("Hello, Student!");
    }
}

2. Compile:

javac Student.java

3. Run:

java Student

Running Java Programs with Packages

If your Java file has a package declaration:

package com.example;
public class MyClass {
    // class content
}

Compile and run:

javac com/example/MyClass.java
java com.example.MyClass

Common Issues and Solutions

Error: "Could not find or load main class"

  • Check if .class file exists in the same directory
  • Ensure class name matches exactly (case-sensitive)
  • Make sure you're in the correct directory

Error: "javac is not recognized"

  • Java JDK is not installed or not in PATH
  • Install JDK and set JAVA_HOME environment variable

Error: "Public class must be in a file with the same name"

  • File name must match the public class name exactly
  • Example: public class Test must be in Test.java
